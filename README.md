# Front-end SIGOS
## ReactJS + Vite
* Javascript Vanilla;
* HTML+ CSS

## como executar:
1. clone o repositório;
2. abra o terminal na raiz do projeto;
3. no terminal, execute:
   ```
   npm install
   ```
4. após a instalação das dependências, execute:
   ```
   npm run dev
   ```
5. o terminal mostrará o link de acesso.
